This folder contains images for the Asteroids game.

To use real images in your game:
1. Add spaceship.png - A top-down view of a spaceship
2. Add asteroid_large.png, asteroid_medium.png, and asteroid_small.png - Images of asteroids in different sizes
3. Optionally add explosion.png for explosion animations

You can find free game assets on sites like:
- OpenGameArt.org
- Kenney.nl
- Itch.io (free assets section)
- GameDev Market (free section)
